LS-PAGGE provides non-expert users with a relatively simple way of creating and modifying level set objects for the use in simulation methods such as computational fluid dynamics (CFD), the discrete element method (DEM), or the phase field method (PFM). The motivation behind LS-PAGGE is that level sets (LS) or signed distance functions (SDF) can be incredibly efficient to describe tooling geometries or complex particle shapes in DEM. However, manipulating level sets is also non-trivial, creating a rather high entry barrier for those that do not have the time to become level-set experts themselves. LS-PAGGE thus provides a wide range of default shapes as well as import methods for images, meshes, and clumps to create level sets. Similarly, the level sets can be adjusted and exported for e.g. (V)LS-DEM simulation in YADE.

The repository can be found here: [DLHvdH/ls-pagge](https://gitlab.com/DLHvdH/ls-pagge)
